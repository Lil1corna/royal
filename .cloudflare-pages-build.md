# Cloudflare Pages Build Configuration

## В дашборде Cloudflare Pages используйте эти настройки:

### Build Settings

```
Framework preset: Next.js
Build command: npx @cloudflare/next-on-pages
Build output directory: .vercel/output/static
Root directory: (leave empty)
```

### Environment Variables

Добавьте в Settings → Environment variables → Production:

```
NODE_VERSION=18
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGci...
SUPABASE_SERVICE_ROLE_KEY=eyJhbGci...
NEXT_PUBLIC_SITE_URL=https://your-project.pages.dev
```

### Important Notes

1. **Node Version**: Обязательно установите `NODE_VERSION=18` или `NODE_VERSION=20`
2. **Build Command**: Используйте `npx @cloudflare/next-on-pages` вместо `npm run build`
3. **Output Directory**: `.vercel/output/static` (не `.next`)

## После деплоя

Если всё равно ошибки:
1. Удалите проект в Cloudflare Pages
2. Создайте заново с этими настройками
3. Убедитесь, что используется правильный build command
